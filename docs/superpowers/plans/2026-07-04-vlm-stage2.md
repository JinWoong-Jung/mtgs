# VLM Stage-2 (graph-text + graph-token) 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stage-1이 끝난 MTGS+graph를 고정한 채, VSGaze에서 그래프 특징을 offline 추출해 저장하고, 그 캐시로 Qwen3-VL-8B를 LoRA 미세조정(graph-text B/D/E + graph-token C)한 뒤 그래프와 soft-blend하여 성능을 올린다.

**Architecture:** `mtgs/`(그래프)는 그대로 두고, 형제 패키지 `vlm/`을 새로 만든다. 의존 방향은 한쪽(`vlm` → `mtgs`)이며 `mtgs`는 `vlm`을 절대 import하지 않는다. `mtgs` core 수정은 `GazeGraphBlock.forward`에 **플래그로 켜고 끄는 `_feat` stash 한 블록**이 유일하다(기본 off → 학습/추론 숫자 무영향). 그래프 실행은 offline 추출(Task 3) 한 번뿐이고, 이후 학습·평가는 디스크 캐시(PNG 오버레이 + `.pt` 그래프캐시 + manifest)만 읽는다.

**Tech Stack:** PyTorch, PyTorch Lightning(기존 MTGS), `transformers 5.8.1`(Qwen3-VL), `peft`(LoRA), `accelerate`, `omegaconf`, `wandb`, SLURM(rtx6000).

## Global Constraints

- **core 무수정 원칙:** `mtgs/` 하위는 `GazeGraphBlock`의 export 훅(플래그 기본 `False`)을 제외하고 **단 한 줄도 바꾸지 않는다.** ([[mtgs-core-vs-origin]] 원칙 준수 — 논문에서 "그래프 front-end는 원본, VLM은 별개 패키지"로 주장 가능해야 함)
- **패키지 배치:** 새 코드는 전부 `MTGS/vlm/` (형제 패키지). 실행은 `MTGS/`에서 `python -m vlm.<module>`.
- **peer 원본 준수:** 로직은 `/home/jinwoongjung/peer_vlm/sgg/`를 기준으로 이식. 이식 시 **경로 하드코딩(`/home/sujungoh/...`, `ROOT`) 제거 → 내 `config.yaml` 경유**, `social_graph.*` 키(구 아키텍처)는 사용하지 않음.
- **모델:** `Qwen/Qwen3-VL-8B-Instruct` (HF hub, 자동 다운로드). LoRA r=16, α=32, LM-only `{q,k,v,o,gate,up,down}_proj`, vision frozen, dropout 0.05, lr 1e-4.
- **그래프 캐시 orientation:** 블록이 계산한 `lah_mat` 등을 **가공 없이 raw로 stash**하고, peer의 export/consumer 코드를 **그대로** 이식하여 orientation 자기일관성을 유지한다(peer가 eval로 검증한 규약: 소비 측에서 `lah[j,i]` = "i→j").
- **지표:** W&B에는 그래프와 동일하게 `compute_metrics.compute()`가 뱉는 `Dist, AP_IO, F1_LAH(PP), F1_LAEO(PP), F1_LAH, F1_LAEO, AP_SA`를 기록. 프로젝트 `MTGS`, entity `gaze-social`, group `vlm-stage2`.
- **sid 정렬(critical):** `graph_export`와 `data_prep`의 enumeration DataLoader는 **반드시 `num_workers=0`**. VSGaze `__getitem__`이 people-subset 선택에 numpy/python `random`을 쓰는데 PyTorch가 이를 worker별로 재시드하지 않아, `num_workers>0`이면 같은 sid가 서로 다른 샘플을 가리켜 캐시↔오버레이가 어긋난다(검증됨: nw>0 → bbox 0.588 drift). 병렬화는 `--nshards`(프로세스 분할)로만.
- **파이썬 인터프리터(검증용):** `/home/jinwoongjung/.conda/envs/mtgs/bin/python`. 임포트 검증 시 `XFORMERS_DISABLED=1` 사용.

## 그래프 캐시 스키마 (canonical — 모든 downstream이 참조)

`vlm/graph_export.py`가 sid(`sampleNNNNNN`)별로 저장하는 dict. 좌표는 정규화 [0,1], center 프레임 기준(별도 표기 제외). `N`=슬롯 수, `De`=edge_dim, `Tl`=N+2, `T`=temporal_context 프레임 수.

```
{
  # --- 그래프 logits (center) : [i,j] raw (블록 출력 그대로) ---
  "lah_logits":  (N,N) fp16,   "laeo_logits": (N,N),  "sa_logits": (N,N),
  "null_in_logits": (N,),      "null_out_logits": (N,),
  # --- per-frame (temporal 텍스트용) ---
  "lah_logits_frames": (T,N,N), "laeo_logits_frames": (T,N,N), "sa_logits_frames": (T,N,N),
  "null_in_frames": (T,N),      "null_out_frames": (T,N),
  # --- 임베딩 (token 경로 C + HGR용, center) ---
  "v_src": (N,De), "v_tgt": (Tl,De),
  "edge_pp": (N,N,De), "edge_null_in": (N,De), "edge_null_out": (N,De),
  "align": (N,N),
  # --- 기하 (graph-text용, center) ---
  "gaze_vecs": (N,2), "gaze_point": (N,2), "overlap": (N,N), "head_bboxes": (N,4),
  # --- GT + mask ---
  "lah_gt": (N,N), "laeo_gt": (N,N), "sa_gt": (N,N), "inout_gt": (N,),
  "person_mask": (N,) bool, "vis_mask": (N,) bool, "num_persons": int,
}
```

키 이름은 peer의 `injection.py`/`eval.py`가 읽는 이름과 정확히 일치해야 한다(`gaze_vecs`, `overlap`, `gaze_point`, `head_bboxes`, `edge_pp`, `v_src`, `v_tgt`, `edge_null_in/out`, `*_logits`, `*_logits_frames`, `vis_mask`, `person_mask`).

def make_cfg(split, *, task="test", use_graph=True):

핵심 본문(추출 루프):
```python
import argparse, random
from pathlib import Path
import numpy as np
import torch
import torch.multiprocessing as _mp
_mp.set_sharing_strategy("file_system")   # "resize storage not resizable" 회피
from torch.utils.data import DataLoader
from tqdm import tqdm
from mtgs.networks.models import MTGSModel
from mtgs.train.dataset import build_dataset
from vlm.cfg import make_cfg
from vlm._pairmat import pair_vector_to_matrix

# 내 build_dataset 은 test_split 을 무시한다 → split 은 setup(stage)+attr 로 선택.
STAGE   = {"train": "fit",   "val": "validate",     "test": "test"}
ATTR    = {"train": "train_dataset", "val": "val_dataset", "test": "test_dataset"}
DLOADER = {"train": "train_dataloader", "val": "val_dataloader", "test": "test_dataloader"}


def _to_dev(o, d):
    if torch.is_tensor(o): return o.to(d)
    if isinstance(o, dict): return {k: _to_dev(v, d) for k, v in o.items()}
    if isinstance(o, (list, tuple)): return type(o)(_to_dev(v, d) for v in o)
    return o


@torch.no_grad()
def main():
    ap = argparse.ArgumentParser()
    ap.add_argument("--split", default="val", choices=["train", "val", "test"])
    ap.add_argument("--ckpt", required=True)
    ap.add_argument("--out", required=True)
    ap.add_argument("--batch_size", type=int, default=4)
    ap.add_argument("--num_workers", type=int, default=4)
    ap.add_argument("--limit", type=int, default=0)
    ap.add_argument("--save_every", type=int, default=4000)
    args = ap.parse_args()

    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    cfg = make_cfg(args.split); cfg.test.batch_size = args.batch_size

    # sid = enumeration index. data_prep(Task 5)와 sid 정렬을 위해 build_dataset 전에
    # 동일 시드 고정 + train split 은 stochastic 증강 비활성화(eval transform)로 맞춘다.
    torch.manual_seed(0); np.random.seed(0); random.seed(0)
    data = build_dataset(**cfg); data.setup(STAGE[args.split])
    if args.split == "train":
        eval_tf = data.val_dataset.datasets[0].transform
        for sub in data.train_dataset.datasets:
            sub.split = "val"; sub.transform = eval_tf
    ds = getattr(data, ATTR[args.split])
    collate = getattr(data, DLOADER[args.split])().collate_fn
    loader = DataLoader(ds, batch_size=args.batch_size, shuffle=False,
                        num_workers=args.num_workers, collate_fn=collate)

    model = MTGSModel(cfg)
    ck = torch.load(args.ckpt, map_location="cpu", weights_only=False)
    miss, unexp = model.load_state_dict(ck["state_dict"], strict=False)
    print(f"[export] ckpt epoch={ck.get('epoch')} missing={len(miss)} unexpected={len(unexp)}", flush=True)
    model.eval().to(device)
    model.model.gaze_graph_block.export_features = True

    autocast = torch.autocast("cuda", dtype=torch.bfloat16) if device.type == "cuda" \
        else torch.autocast("cpu", enabled=False)
    out_path = Path(args.out); out_path.parent.mkdir(parents=True, exist_ok=True)
    cache = {}; sample_idx = 0

    for batch in tqdm(loader, desc=f"export:{args.split}"):
        batch = _to_dev(batch, device)
        with autocast:
            model(batch)
        f = model.model.gaze_graph_block._feat
        B, T, N = f["lah_mat"].shape[:3]; c = T // 2; De = f["v_src"].shape[-1]
        E = f["E"]                                             # (B,T,N,N+2,De)
        lah_c = f["lah_mat"][:, c].float().cpu()               # (B,N,N)
        sa_c  = f["sa_mat"][:, c].float().cpu()
        # LAEO: lah_min 모드면 laeo_mat=None → min(lah, lah^T) (logit space, 대칭)
        if f["laeo_mat"] is None:
            laeo_all = torch.minimum(f["lah_mat"], f["lah_mat"].transpose(-1, -2))
        else:
            laeo_all = f["laeo_mat"]
        laeo_c = laeo_all[:, c].float().cpu()

        # GT ([i,j]="i looks at j"): lah reverse=True (peer 규약)
        lah_gt  = pair_vector_to_matrix(batch["lah_labels"][:, c, :],  N, reverse=True).cpu()
        laeo_gt = pair_vector_to_matrix(batch["laeo_labels"][:, c, :], N).cpu()
        sa_gt   = pair_vector_to_matrix(batch["coatt_labels"][:, c, :], N).cpu()
        inout_gt = batch["inout"][:, c, :].cpu()
        bbc = batch["head_bboxes"][:, c].float().cpu()         # (B,N,4)
        nvp = batch["num_valid_people"][:, c].long().cpu()     # (B,)
        ar = torch.arange(N).view(1, N)
        pmask = ar >= (N - nvp.view(-1, 1))                    # (B,N) 뒤쪽 valid
        area = (bbc[..., 2]-bbc[..., 0]).clamp_min(0) * (bbc[..., 3]-bbc[..., 1]).clamp_min(0)
        vis = pmask & (area > 1e-4)

        for b in range(B):
            sid = f"sample{sample_idx + b:06d}"
            cache[sid] = {
                "lah_logits": lah_c[b].half(), "laeo_logits": laeo_c[b].half(), "sa_logits": sa_c[b].half(),
                "null_in_logits": f["null_in"][b, c].float().cpu().half(),
                "null_out_logits": f["null_out"][b, c].float().cpu().half(),
                "lah_logits_frames":  f["lah_mat"][b].float().cpu(),
                "laeo_logits_frames": laeo_all[b].float().cpu(),
                "sa_logits_frames":   f["sa_mat"][b].float().cpu(),
                "null_in_frames":  f["null_in"][b].float().cpu(),
                "null_out_frames": f["null_out"][b].float().cpu(),
                "v_src": f["v_src"][b, c].half().cpu(), "v_tgt": f["v_tgt"][b, c].half().cpu(),
                "edge_pp":       E[b, c, :, :N, :].half().cpu(),
                "edge_null_in":  E[b, c, :, N,  :].half().cpu(),
                "edge_null_out": E[b, c, :, N+1, :].half().cpu(),
                "align":   f["align"][b, c].half().cpu(),
                "overlap": f["overlap"][b, c].half().cpu(),
                "gaze_vecs":  f["gaze_vecs"][b, c].float().cpu(),
                "gaze_point": f["gaze_point"][b, c].float().cpu(),
                "head_bboxes": bbc[b],
                "lah_gt": lah_gt[b], "laeo_gt": laeo_gt[b], "sa_gt": sa_gt[b], "inout_gt": inout_gt[b],
                "person_mask": pmask[b], "vis_mask": vis[b], "num_persons": int(pmask[b].sum()),
            }
        sample_idx += B
        if args.limit and sample_idx >= args.limit: break
        if len(cache) and len(cache) % args.save_every < B: torch.save(cache, out_path)

    torch.save(cache, out_path)
    print(f"[export] saved {len(cache)} samples -> {out_path}", flush=True)


if __name__ == "__main__":
    main()
```

### Task 10: `scripts/train_vlm.sh` — MODE 전환 sbatch

**Files:**
- Create: `scripts/train_vlm.sh`

**Interfaces:**
- Consumes: Task 3/5/8/9의 CLI.
- Produces: `sbatch scripts/train_vlm.sh` 실행 시 상단 변수(`MODE`, `CHECKPOINT`, `EXP_NAME`, `SPLIT`, `EXPERIMENT`)에 따라 export/overlays/graphtext/token/eval 단계 실행.

- [ ] **Step 1: 스크립트 작성**

기존 `scripts/train_vsgaze.sh`의 SLURM 헤더(rtx6000, conda activate mtgs, `cd $MTGS`, `XFORMERS_DISABLED=1` 등)를 참고해 헤더를 맞추고, 본문:
```bash
#!/bin/bash
#SBATCH --job-name=vlm_stage2
#SBATCH --partition=rtx6000
#SBATCH --gres=gpu:1
#SBATCH --output=scripts/logs/vlm_%j.out
# (기존 train_vsgaze.sh 헤더의 나머지 옵션 복사)

set -e
source ~/.bashrc && conda activate mtgs
cd /home/jinwoongjung/MTGS
export XFORMERS_DISABLED=1

# ── 설정 ──────────────────────────────────────────────
MODE=${MODE:-graphtext}        # export | overlays | graphtext | token | eval
SPLIT=${SPLIT:-train}
CHECKPOINT=${CHECKPOINT:-experiments/V14.5/train/checkpoints/best.ckpt}
EXPERIMENT=${EXPERIMENT:-B_graphtext}      # 실험 이름(=out 디렉토리/wandb name)
BLIND=${BLIND:-0}              # 1 → D (answer-blind)
UPWEIGHT=${UPWEIGHT:-1}        # >1 → E (graph-wrong 업샘플)
CACHE=results/vlm_cache
OUT=experiments/vlm/$EXPERIMENT

mkdir -p $CACHE scripts/logs

case $MODE in
  export)
    python -m vlm.graph_export --split $SPLIT --ckpt $CHECKPOINT \
      --out $CACHE/vlmgraph_${SPLIT}.pt --batch_size 4 ;;
  overlays)
    python -m vlm.data_prep overlays --split $SPLIT --out $CACHE/overlays \
      --manifest $CACHE/manifest_${SPLIT}.jsonl --gtmeta $CACHE/gtmeta_${SPLIT}.pt ;;
  graphtext)
    GRAPHTEXT_BLIND=$BLIND GRAPH_WRONG_UPWEIGHT=$UPWEIGHT \
    python -m vlm.train nograph \
      --manifest $CACHE/manifest_train.jsonl --overlay_dir $CACHE/overlays/train \
      --graph_feats $CACHE/vlmgraph_train.pt --out $OUT \
      --val_manifest $CACHE/manifest_val.jsonl --val_overlay_dir $CACHE/overlays/val \
      --val_gtmeta $CACHE/gtmeta_val.pt --val_graph_feats $CACHE/vlmgraph_val.pt \
      --epochs 2 --bs 2 --accum 8 --lr 1e-4 --wandb_name $EXPERIMENT ;;
  token)
    python -m vlm.train token \
      --manifest $CACHE/manifest_train.jsonl --overlay_dir $CACHE/overlays/train \
      --graph_feats $CACHE/vlmgraph_train.pt --out $OUT \
      --val_manifest $CACHE/manifest_val.jsonl --val_overlay_dir $CACHE/overlays/val \
      --val_gtmeta $CACHE/gtmeta_val.pt --val_graph_feats $CACHE/vlmgraph_val.pt \
      --epochs 2 --bs 8 --lr 1e-4 --wandb_name $EXPERIMENT ;;
  eval)
    python -m vlm.eval nograph --ckpt $OUT/final \
      --manifest $CACHE/manifest_${SPLIT}.jsonl --overlay_dir $CACHE/overlays/$SPLIT \
      --gtmeta $CACHE/gtmeta_${SPLIT}.pt --graph_feats $CACHE/vlmgraph_${SPLIT}.pt \
      --preds_out $CACHE/preds_${EXPERIMENT}_${SPLIT}.pt
    python -m vlm.eval blend --feat $CACHE/vlmgraph_${SPLIT}.pt \
      --pvlm $CACHE/preds_${EXPERIMENT}_${SPLIT}.pt --alphas 0,0.25,0.3,0.5,1.0 ;;
  *) echo "unknown MODE=$MODE"; exit 1 ;;
esac
```
